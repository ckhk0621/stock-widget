Stock Widget - WordPress Embed Test

Test 1: Script Embed with Mock Data (Recommended)

This simulates how the widget will appear in WordPress with mock data.

Instructions for WordPress

Copy this exact code into a Custom HTML block in WordPress:

<div id="stock-widget"></div>
<script>
  window.stockWidgetConfig = {
    symbol: 'MIMI',
    useMock: true,
    theme: 'light'
  };
</script>
<script type="module" crossorigin src="https://stock-widget-five.vercel.app/stock-widget.js"></script>
<link rel="stylesheet" href="https://stock-widget-five.vercel.app/stock-widget.css">

Configuration Options

  • symbol: Stock ticker (e.g., 'MIMI', 'AAPL', 'GOOGL')
  • useMock: Use mock data (true = demo data, false = real API)
  • theme: Color scheme ('light' or 'dark')
